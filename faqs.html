Frequently Asked Questions

What is TEI?

See Oxford University’s explanation for an answer to this question.

Where do I find TEI-encoded XML of plays?

Here is a list of some sources:

  • Oxford Text Archive
  • Folger Digital Texts
  • Perseus Digital Library
  • Dramacode
  • Deutches Text Archiv
  • There are more sources listed on the TEI Consortium’s website

You can also create your own!

Why isn’t my file uploading? Why isn’t the Prompter working?

There are a few common reasons why this might have happened:

  • The file is too large. If this is the case, split the file into two separate TEI documents and upload each separately.
  • The file must be in TEI format. Check the raw XML to make sure that this is the case and edit if necessary.
  • The play does not contain acts. If the play you are uploading is not divided into acts, then the Prompter will not be able to work on it at the moment. The ability to prepare parts for plays without acts will be added in a future update.

If you have a different issue, please contact the developer.

Where can I find out more about this stuff?

Read these two books:

  • Palfrey, Simon, and Tiffany Stern, Shakespeare in Parts (Oxford: Oxford University Press, 2007)
  • Stern, Tiffany, Rehearsal from Shakespeare to Sheridan (Oxford: Oxford University Press, 2000)
